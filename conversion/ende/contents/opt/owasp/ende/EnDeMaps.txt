#                 http://en.wikipedia.org/wiki/Character_entity_reference
#           NCR - numeric character references
#                 http://en.wikipedia.org/wiki/Numeric_character_reference
#           UCS - universal character set
#                 http://en.wikipedia.org/wiki/Universal_Character_Set
#                 BOM - Byte Order Mark
#                 BMP - Basic Multilingual Plane
#                 UCS-2, a single code value between 0 and 65535
#                 UTF-16, an extension of UCS-2, to represents code points
#                         outside the BMP
#                 UCS-4, a single code value between 0 and (theoretically)
#                         hexadecimal 7FFFFFFF
#                         UCS-4 thereby permits a binary representation of
#                         every code point in the UCS
#           ==> articles about Unicode will mistakenly refer to  UCS-2  as
#               "UCS-16". UCS-16 does not exist.
#
#           "Character 65533 (FFFD hexadecimal) is the last valid character in
#            UCS-2. 65534 (FFFE hexadecimal) is unassigned and reserved as the
#            byte-swapped version of  ZERO WIDTH NON-BREAKING SPACE  for byte-
#            order detection purposes. 65535 (FFFF hexadecimal) is unassigned."
#

#            range within the CP-1252 charset. These entities permit the
#            characters to be denoted in a platform-independent manner."
#            CP-1252 is same or similar to ANSI see also http://www.goascii.de/
