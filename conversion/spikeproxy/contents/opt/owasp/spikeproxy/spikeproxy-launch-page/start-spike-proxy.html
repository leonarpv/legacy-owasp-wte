You will interact with Spike Proxy with a browser.

First you must make sure that the browser is using the Spike Proxy
by changing the proxy settings to http://127.0.0.1:8068

After the proxy is setup correctly, navigate to http://spike/ to view the GUI in your browser.

Spike Proxy logs to /opt/owasp/spikeproxy/logs/ in a file named spikeproxy-[date&time].log

Spike Proxy's cache directory is located at /var/cache/spikeproxy.
